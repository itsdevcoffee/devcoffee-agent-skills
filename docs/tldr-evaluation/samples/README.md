# TLDR Evaluation Samples

This directory contains individual evaluation samples of the `/tldr` command in real-world usage.

## Sample Format

Each file: `NNN-brief-description.md`

Example:
- `001-code-implementation.md`
- `002-research-summary.md`
- `003-bugfix-analysis.md`

## Contents

Each sample file includes:
- Original Claude message (what was summarized)
- TLDR output (the summary)
- Detailed scoring breakdown (Completeness, Conciseness, Actionability, Accuracy)
- Analysis and recommendations
- Optional user feedback
